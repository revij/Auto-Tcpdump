## Setup (Ubuntu and Debian)
``` sudo apt-get install dos2unix``` 

``` sudo apt-get install tcpdump```

``` sudo apt-get install screen```

``` chmod +x tcpdump.sh ``` 

``` dos2unix tcpdump.sh```

``` screen bash tcpdump.sh```
